Nós escolhemos fazer o nosso formulário relacionado a  livros no qual pedimos qual seu livro favorito, gênero e algumas informações adicionais como nome, e-mail, e no final colocamos uma caixa de observação caso a pessoa queira citar algo relacionado ao livro.
Para criar o formulário utilizamos a base simples da qual o professor explicou em aula, criamos um arquivo index. com o corpo do formulário, css no qual mudamos cor, espaçamento, entre outros, o script para salvar os dados, e linkamos uma página de obrigado para quando os dados forem enviados.
