Naomy e Thabata info 1A


A gente escolheu o tema de livros, onde fazemos perguntas em relação a isso, com o livro favorito, autor etc. A gente uso do formulário que o professor havia passado na aula passada, contando html, onde aprendemos a usar "form", e dentro adicionamos informações com as tags necessárias para cria-lo, como input e label, também usamos do css, para estilizar a pagina, além de criar um botão que altera as cores para modo escuro e claro. Utilizamos do javaScript, para que o botão funcione e também para salvar os dados das resposta no LocalStorage.
