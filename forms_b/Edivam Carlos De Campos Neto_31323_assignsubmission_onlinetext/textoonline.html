Fizemos um site da empresa IBJE, uma organização que faz sensos sobre academias pelo país. No HTML, colocamos informações, como: nome, peso, altura, data de nascimento, idade e sexo. Após você colocar as suas informações no site, você é levado para uma página de agradecimento, já que seria uma empresa sem fins lucrativos. Após pesquisas usamos FadeIn para deixar a página, uma animação no final da tela por questão estética, usamos "radio" para escolher o sexo da pessoa que está respondendo, "exp" para experiência na academia e o "LocalStorage" para salvar os dados. Na página de agradecimentos colocamos o CSS e o JavaScript juntos para otimizar o código e facilitar o entendimento.
