O tema escolhido para o formulário de satisfação usado foi o atendimento de Hospital, o nome escolhido é fictício, o site questiona alguns dados pessoais e logo depois, a avaliação que o cliente deseja fazer para uma pesquisa de satisfação com o propósito de melhorar cada vez mais o atendimento e infraestrutura do hospital.

O site foi usado CSS e JavaScript, alguns termos técnicos foram:

"radio" que são os "botões redondos" para indicar gênero, qual o profissional, avaliação do atendimento e infraestrutura.

"checkbox" para aceitar os termos

Uma "textarea" para que o usuário possa adicionar outras informações desejadas.

"option value" para adicionar opções dos hospitais da rede para o usuário selecionar em qual hospital de qual cidade deu entrada.
